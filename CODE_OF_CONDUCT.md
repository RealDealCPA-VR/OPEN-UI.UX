# Code of Conduct

OpenCodex adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The full text — including the standards, scope, and enforcement responsibilities — lives at that link and is incorporated here by reference.

## Summary

Participation in this project should be a respectful, welcoming experience for everyone, regardless of background or identity. We expect contributors to act in good faith, communicate constructively, and focus on the work.

The Contributor Covenant defines the specific expectations and the kinds of conduct that are out of bounds. Read the linked document in full before participating — by contributing, you agree to be bound by it.

## Scope

This Code applies in all project spaces: the repository, issues, pull requests, discussions, chat channels, and any public space where someone is representing the project.

## Reporting

If you experience or witness conduct that violates this Code, report it to the project maintainers via the security contact in [SECURITY.md](./SECURITY.md). Reports will be reviewed and investigated promptly and fairly. The reporter's identity will be kept confidential.

Maintainers who do not follow or enforce the Code in good faith may face consequences determined by the project leadership.
